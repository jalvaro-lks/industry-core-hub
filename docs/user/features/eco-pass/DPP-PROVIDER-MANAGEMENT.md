# Passport Consumption & Visualization

> Consume and visualize a published Digital Product Passport via QR or identifier.

---

## Purpose

This feature allows users to access and view Digital Product Passports that have been published in the data space, using either a QR code or a unique identifier.

---

## Features
- Scan a QR code or enter an identifier to retrieve a passport.
- View all details and metadata of the passport.
- Visual representation of passport data.

---

## Usage Guide

1. Access the Passport Consumption & Visualization section.
2. Scan a QR code or enter the passport identifier.
3. Review the displayed passport information.

## Tips
- Ensure the passport is published and accessible in the data space.
- Use the latest browser for best QR scanning support.

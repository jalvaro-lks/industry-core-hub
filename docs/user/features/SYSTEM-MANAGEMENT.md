# System Management

> Connect, register, and manage systems related to your Industry Core Hub.

---

## Purpose

The System Management feature allows you to integrate and oversee external systems that interact with your Industry Core Hub instance.

---

## Features
- Register new systems.
- View and edit system details.
- Monitor system status and connectivity.
- Remove or update system integrations.

---

## Usage Guide

1. Access the System Management section from the main navigation.
2. Click “Add System” to register a new system.
3. Fill in the required details and save.
4. View, edit, or remove systems as needed.

## Tips

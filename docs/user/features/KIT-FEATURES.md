# KIT Features

> Manage and configure the features of each KIT integrated into the Industry Core Hub.

---

## Purpose

The KIT Features section allows you to enable or disable specific features for each KIT available in the Industry Core Hub. This helps you customize your application experience and control which functionalities are accessible from the frontend.

---

## Features
- Browse all available and upcoming KITs.
- View the status of each KIT (available, coming soon, etc.).
- Access documentation for each KIT and its features.
- Enable or disable features, which will then appear (or not) in the ICHub frontend navigation.
- Quickly navigate to the screen of any enabled feature.

---

## Usage Guide

### Main Screen
1. Access the KIT Features section from the main navigation.
2. Browse through the carousel of KITs.
3. Click on a KIT to view its available features and status.
4. Use the toggles to enable or disable specific features.
5. Click on a feature to go directly to its dedicated screen.

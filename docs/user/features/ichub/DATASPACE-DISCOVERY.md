# Dataspace Discovery

> Consume Catalog and Serialized Parts published by your contacts.

---

## Purpose

Dataspace Discovery enables you to find and access data products shared by other participants in your network.

---

## Features
- Search for Catalog and Serialized Parts published by contacts.
- View detailed information about available data products.
- Initiate interactions or requests for data.

---

## Usage Guide

1. Access the Dataspace Discovery section.
2. Search or browse available data products.
3. View details and interact as needed.

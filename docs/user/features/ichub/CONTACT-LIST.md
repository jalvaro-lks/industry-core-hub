# Contact List

> Define and manage data space participants (Name and BPNL).

---

## Purpose

The Contact List feature lets you manage the participants you interact with in the data space.

---

## Features
- Add, edit, or remove contacts.
- Store participant names and BPNL identifiers.
- Use contacts for sharing and discovery features.

---

## Usage Guide

1. Access the Contact List section.
2. Add new contacts with name and BPNL.
3. Edit or remove contacts as needed.

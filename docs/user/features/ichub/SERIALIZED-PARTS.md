# Serialized Parts

> Create, assign to Catalog Parts, register, and publish Serialized Parts.

---

## Purpose

Serialized Parts allow you to track and manage individual components, linking them to Catalog Parts.

---

## Features
- Create new Serialized Parts.
- Assign Serialized Parts to existing Catalog Parts.
- Register and publish Serialized Parts in the data space.

---

## Usage Guide

1. Access the Serialized Parts section.
2. Click “Create Serialized Part” and fill in the details.
3. Assign to a Catalog Part.
4. Register and publish.

## Tips
- Always link Serialized Parts to the correct Catalog Part.
- Use unique identifiers for traceability.
